Import from a Photo

      
Take a photo of any recipe, open Claude, and use this prompt. Then paste the JSON in the JSON tab.

      
📷 Photo Prompt

        
Please extract this recipe from the photo and return ONLY a JSON object (no markdown, no extra text):
 
{
  "name": "Recipe Name",
  "serves": 4,
  "sourceUrl": "",
  "tags": [],
  "ingredients": [
    { "name": "ingredient (no prep instructions — just the item)", "quantity": 1, "unit": "g", "category": "Produce", "pantry": false }
  ],
  "steps": [
    { "text": "Step description. Any measurements in step text must use [[qty|unit]] markers e.g. 'Add [[200|g]] flour'.", "timerSeconds": 0 }
  ]
}
 
IMPORTANT — Ingredient prep handling:
- The "name" field must contain ONLY the raw ingredient with no prep verbs (e.g. "yellow onion" not "yellow onion, diced").
- If an ingredient has prep instructions (diced, minced, sliced, chopped, grated, julienned, etc.), collect ALL of them and create a FIRST step in the "steps" array titled "Prep" that lists every ingredient needing prep, e.g.: "Prep: Dice 1 yellow onion. Mince 3 cloves garlic. Slice 2 red peppers." timerSeconds for this step should be 0.
- The remaining steps follow as normal after the prep step.
 
IMPORTANT — Step text measurements:
- Wrap any ingredient quantities mentioned in steps with [[qty|unit]] markers so they can be auto-scaled when servings change. E.g. "Add [[200|g]] flour and [[100|ml]] milk".
- Only wrap ingredient amounts, NOT cooking times or temperatures.
- The qty must be the amount for the base serving size given above.
 
Category MUST be exactly one of these seven — use the definitions to decide:
- Produce: fresh fruit, vegetables, fresh herbs — e.g. onion, spinach, lemon, basil, garlic
- Bakery: bread, wraps, raw pastry/dough — e.g. sourdough, pita, croissants, pizza base
- Deli: cured/smoked meats, prepared dips, ready-made items — e.g. salami, smoked salmon, hummus, tzatziki
- Butcher: raw meat, raw poultry, raw seafood — e.g. chicken breast, ground beef, salmon fillet, prawns
- Dairy: milk, cheese, yogurt, cream, butter, eggs — e.g. cheddar, Greek yogurt, heavy cream, unsalted butter
- Frozen: anything purchased in frozen state — e.g. frozen peas, ice cream, frozen pastry sheets
- Other: pantry items, canned goods, condiments, dry goods, oils, sauces — e.g. pasta, olive oil, soy sauce, tinned tomatoes, flour, sugar
 
Unit options: g, kg, ml, l, tsp, tbsp, cup, oz, lb, piece, or "" for countable items (e.g. 2 eggs → unit "")
timerSeconds: 0 if no waiting; otherwise total seconds (e.g. 20 min = 1200)
tags: pick 1–4 from — Quick, Vegetarian, Vegan, Gluten-Free, Weekend, Kids Favourite, Comfort Food, Light, Batch Cook, Fish, Pasta, Soup, Salad
sourceUrl: leave as ""
