📷 Photo Prompt

Please extract this recipe from the photo and return ONLY a JSON object (no markdown, no extra text):

{
  "name": "Recipe Name",
  "serves": 4,
  "sourceUrl": "",
  "tags": [],
  "ingredients": [
    { "name": "ingredient", "quantity": 1, "unit": "g", "category": "Produce" }
  ],
  "steps": [
    { "text": "Step description.", "timerSeconds": 0 }
  ]
}

Category MUST be exactly one of these seven — use the definitions to decide:
- Produce: fresh fruit, vegetables, fresh herbs — e.g. onion, spinach, lemon, basil, garlic
- Bakery: bread, wraps, raw pastry/dough — e.g. sourdough, pita, croissants, pizza base
- Deli: cured/smoked meats, prepared dips, ready-made items — e.g. salami, smoked salmon, hummus, tzatziki
- Butcher: raw meat, raw poultry, raw seafood — e.g. chicken breast, ground beef, salmon fillet, prawns
- Dairy: milk, cheese, yogurt, cream, butter, eggs — e.g. cheddar, Greek yogurt, heavy cream, unsalted butter
- Frozen: anything purchased in frozen state — e.g. frozen peas, ice cream, frozen pastry sheets
- Other: pantry items, canned goods, condiments, dry goods, oils, sauces — e.g. pasta, olive oil, soy sauce, tinned tomatoes, flour, sugar

Unit options: g, kg, ml, l, tsp, tbsp, cup, oz, lb, piece, or "" for countable items (e.g. 2 eggs → unit "")
timerSeconds: 0 if no waiting; otherwise total seconds (e.g. 20 min = 1200)
tags: pick 1–4 from — Quick, Vegetarian, Vegan, Gluten-Free, Weekend, Kids Favourite, Comfort Food, Light, Batch Cook, Fish, Pasta, Soup, Salad
sourceUrl: leave as ""
